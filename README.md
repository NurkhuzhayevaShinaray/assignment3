# Online Bookstore, built using Java and PostgreSQL.

## Features
- Add books
- View all books
- Purchase books
- Delete books

## Technologies
- Java (OOP)
- PostgreSQL (Database)

## How to Run
1. Set up the database using the provided SQL script.
2. Configure the database connection in `BookRepository.java`.
3. Compile and run the `BookStoreApp` class.
